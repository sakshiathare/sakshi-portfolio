## ▶️ Run Locally

```bash
# Clone the repository
git clone https://github.com/sakshiathare/sakshi-portfolio.git
cd sakshi-portfolio

# Install dependencies
npm install

# Start development server
ng serve

# Open http://localhost:4200 in your browser
```
